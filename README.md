# User-management
Ce programme propose à l'utilisateur:
- d'entrer ses données (Nom, Prénoms, Téléphone, Adresse, Sexe)
- d'ajouter les données entrées par l'utilisateur dans le fichier "file_m"
- d'afficher le contenu de ce fichier
- de rechecher un terme (exemple: nom)et renvoyer la ligne concernée
- de modifier les contenus du fichier "file_m"
- de supprimer une ligne du contenu de ce fichier
